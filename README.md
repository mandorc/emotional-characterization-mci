# Emotional Characterization of Clinical Interview Transcripts in Spanish and English for the Detection of Mild Cognitive Impairment

This project explores the emotional characterization of clinical interview transcripts in Spanish and English for Mild Cognitive Impairment (MCI) detection.

The workflow includes automatic speech transcription, text segmentation, multilingual emotion analysis, feature extraction, and classification experiments to evaluate whether emotional patterns in language can help distinguish participants with MCI from healthy controls.

The project focuses on comparing emotional representations across languages and analyzing their usefulness as text-based markers for cognitive impairment.
